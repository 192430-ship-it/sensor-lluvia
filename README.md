Monitor en tiempo real de sensor de lluvia con ESP32 y AWS.

## Tecnologias
- ESP32 (sensor)
- AWS EC2 (servidor)
- AWS DynamoDB (base de datos)
- Node.js + PM2
- Telegram Bot (notificaciones)

## Acceso
Dashboard: http://18.118.25.93:5000

## Funciones
- Monitoreo en tiempo real
- Grafica de valores analogicos
- Notificaciones automaticas a Telegram cuando detecta lluvia
